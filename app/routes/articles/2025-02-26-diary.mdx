## [【Laravel】chunk と chunkById を比較してわかったこと](https://kraf.jp/blog/5620)
これは実務で使う機会があったので、参考になりました。

```php
$usersQuery = User::where('point', '>', 100);
$users->chunk(100, function ($users) {
    foreach ($users as $user) {
        $user->update(['point' => 0]);
    }
});
```
